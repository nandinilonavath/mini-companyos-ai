# 🚀 Mini CompanyOS AI

Mini CompanyOS AI is a multi-agent enterprise decision intelligence platform that helps business leaders make strategic decisions through collaborative AI agents.

## ✨ Features

- 👑 CEO Agent
- 📈 Sales Agent
- 💰 Finance Agent
- 👥 HR Agent
- 🧠 Hindsight Memory
- ⚡ Intelligent Model Routing
- 📊 Executive Decision Dashboard
